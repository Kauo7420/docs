## 战术运用建议

### 交战距离
| 距离 | 战术 |
|------|------|
| 超远距离（>500格） | 利用弹道显示，主炮曲射 |
| 中距离（200-500格） | 主炮平射，副炮准备 |
| 近距离（<200格） | 全炮门齐射，准备鱼雷 |

### 弹药选择
- **穿甲弹**：打击敌舰水线以下，造成进水
- **高爆弹**：打击上层建筑、炮塔、烟囱
- **半穿甲弹**：平衡选择，通用性强

### 机动要点
- 保持侧舷对敌，发挥最大火力
- 避免 T 字不利（船头/船尾对敌）
- 利用航速优势保持距离或抢占优势阵位
