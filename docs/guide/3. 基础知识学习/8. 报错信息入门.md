退出（报错）消息，就是玩家在非主动性或非正常退出服务器时，服务器向玩家展示的信息。我们可以分类讨论这些退出（报错）信息：

::: note
以下内容采用了AI分类，请注意辨识
:::

### 服务器与客户端交互异常
+ **数据流终止（客户端消息）**：表示服务器突然停止向客户端发送数据，可能是服务器崩溃或网络中断所致。
+ **Internal Server Error: java.net.Minecraft**：服务器向客户端发送了无法识别的信息，常见于Mod或插件错误。
+ **Internal exception: java.io.IOException: Received string length longer than maximum allowed ([数字]>256)**：客户端发送的信息长度超过了服务器允许的最大值，通常由Mod或恶意输入引起。
+ **Internal Server Error**：服务器内部错误，可能由代码bug或资源冲突导致。

### 客户端行为问题
+ **断开连接**：玩家主动选择了游戏主菜单中的断开连接选项。
+ **非法字符**：客户端尝试发送含有特殊字符的信息，如控制字符或格式代码。
+ **disconnect.spam**：客户端发送消息的速度过快，触发了反垃圾邮件机制。
+ **连接超时**：服务器未能及时收到客户端的心跳信号，可能是因为网络延迟或客户端断线。

### 身份验证与权限问题
+ **你已在异地登录**：玩家的账户在另一地点被登录，导致当前会话被中断。
+ **登录失败**：客户端运行在离线模式，无法通过Mojang的认证服务。
+ **客户端已过期（请使用{版本}）**：客户端版本低于服务器要求，需更新至指定版本。
+ **服务器已过期（我仍在用{版本}）**：服务器版本低于客户端，客户端无法连接。

### 违规与封禁
+ **你已被此服务器封禁**：玩家因违反服务器规则被管理员封禁。
+ **你的IP已被此服务器封禁**：玩家的IP地址因违规行为被封禁。
+ **被管理员踢出游戏**：管理员使用`/kick`命令将玩家移出服务器。

### 游戏规则与限制
+ **服务器不允许飞行**：玩家在禁止飞行的模式下试图飞行。
+ **尝试攻击无效实体 - ？？**：通常由Mod引发，玩家尝试攻击不存在的实体。
+ **非法高度 - ？？**：玩家位置超出游戏允许的高度范围。
+ **非法位置 - ？？**：玩家位置超出游戏允许的坐标范围。
+ **你死了。游戏结束了！**：在极限模式下，玩家死亡后仍尝试加入游戏。

### 系统与资源问题
+ **你挂机太久了！**：玩家长时间没有动作，超过了服务器设置的挂机时间限制。
+ **内存溢出**：服务器内存耗尽，常见于资源密集型Mod或大量玩家同时在线的情况。
+ **无效会话**：客户端与服务器的会话校验失败，可能由于网络问题或客户端状态异常。
+ **非法IP**：客户端尝试使用无效或无法解析的IP地址连接服务器。
