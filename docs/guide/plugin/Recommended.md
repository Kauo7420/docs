## 不推荐的插件
### 基础管理插件
| **插件名称** | **不推荐的理由** |
| :---: | --- |
| CMI | 虽然说功能很多，但是正是因为功能多，导致它自身非常的臃肿，甚至还出现了经典的“解方程”功能，只能说不要的它都加了进去。不仅如此，它的内存占用也是大的一批，基本上都在后台吃饱了，把你后台资源全吃完了。吃内存就算了，它自身还贵的一批。它也被SunLight成为“超级应用程序”。 |
| EssentialsX | 怎么说呢，这个插件很恶俗，把原版的指令全都替换成EssentialsX插件自身的指令了，比如你要用`/tp`指令的话，它就给你调用EssentialsX的`/tp`，而不是Minecraft自身的`/tp`指令，这就是篡改指令系统。当然，它也加入了少部分不需要的功能，比如`/nuke`（释放核弹）和`/tnt`（快速放置一个引燃的TNT）。而且总体来说，这个插件不易上手，功能简陋，篡改指令，只能说是史中史。 |
| PlayerGuild<br/>RandomShop<br/>PlayerAuction<br/>PlayerReturn<br/>CompanionsPlus<br/>AfdianPay<br/>PlayerTop<br/>OfflinePapi<br/>PlayerTitle<br/>PlayerTask<br/>PlayerWarp | 左侧所列插件全部为开发者“米饭”的付费插件，**这些插件均可能存在远程SQL语句执行后门。**2025年3月，有人发现米饭的大部分付费插件均包含SQL执行命令，用于执行从米饭的付费控制服务器下发的SQL语句。这样开发者有能力远程修改、删除安装此插件的服务器上的数据库，甚至在服务器主机远程执行命令，且全程不会提示用户此行为的存在。<br/>详见MineBBS的处理公告：[https://www.minebbs.com/threads/sql.35470/](https://www.minebbs.com/threads/sql.35470/) |
