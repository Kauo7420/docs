## 为什么要认识文件结构？
!!废话!!

当然是为了扫盲，扫除一些小白服主、菜鸟服主、入门服主的知识盲区，这是最基础的知识。

此教程将教你认识Minecraft服务器的文件结构，了解不同的文件/文件夹的作用，以便更好的架设和维护服务器。

## 文件结构详解
一般来说，服务器的根目录都有如下文件/文件夹

| 文件图标 | **文件/文件夹名称** | **描述/作用** |
| :---: | :---: | :---: |
| 📂 | assets | 存放游戏资源的文件夹（用途不明，不建议乱动） |
| 📂 | .cache<sup>[Paper及其分支特有]</sup> | <font style="color:rgb(44, 44, 54);">存储临时文件或缓存数据的地方，比如下载的资源或最近使用的数据</font> |
| 📂 | cache<sup>[Bukkit服特有]</sup> | 存放原版（Vanilla）服务端核心的地方 |
| 📂 | crash-reports | 存放服务器崩溃日志的地方，当服务器出现常规崩溃、网络协议崩溃、区块文件崩溃时，就会生成一份崩溃报告存储在这里面 |
| 📂 | config<sup>[Paper及其分支特有]</sup> | <font style="color:rgb(44, 44, 54);">用于保存Paper及其分支服务端的服务器的配置文件</font> |
| 📂 | libraries | 按[Maven](https://en.wikipedia.org/wiki/Apache_Maven)公约包装的各种Java库（没有POM文件），如果不知道作用就不要乱动，否则可能会造成服务器崩溃！ |
| 📂 | logs | 存放日志的地方，你服务器的控制台所展示的信息及其之前的信息，全都会按照时间分开来存放再次文件夹 |
| 📂 | plugins<sup>[Bukkit服特有]</sup> | 储存插件jar文件的文件夹<br/>如果你要安装插件，那么就把插件jar文件丢这里面，然后重启服务器 |
| 📂 | version | 存放MC游戏版本的地方，每个版本文件夹中包含了该版本的MC服务端程序及其相应的依赖项 |
| 📂 | world | 主世界文件夹<br/>此文件夹名称可以在[server.properties](https://www.yuque.com/simpfox/mcserverhelp/server.properties)内的level-name一项修改 |
| 📂 | world_nether | 下界世界文件夹 |
| 📂 | world_the_end | 末地世界文件夹 |
| 📄 | .console_history<sup>[Linux操作系统特有]</sup> | 记录通过控制台输入命令历史的文件 |
| 📝 | banned-players.json | 储存被封禁玩家名单数据的文件<br/>使用`/ban`指令封禁玩家后都将储存在这里 |
| 📝 | banned-ips.json | 储存被封禁玩家IP名单数据的文件<br/>使用`/ban-ip`指令封禁玩家后都将储存在这里 |
| 📝 | bukkit.yml<sup>[Bukkit服特有]</sup> | 一个用于配置 CraftBukkit 或其衍生服务器软件<sup>（如 Spigot 或 Paper）</sup>的高级设置的文件 |
| 📝 | _commands.yml_<sup>[Bukkit服特有]</sup> | 一个包含 CraftBukkit 服务器的自定义别名和命令方块覆盖的配置文件<br/>_**这玩意你是用不到的，当摆设就行了**_ |
| 📝 | eula.txt | 一个用于确认服主是否同意并愿意遵守由Mojang设定的Eula（最终用户许可协议）的文件 |
| 📝 | _help.yml_<sup>[Bukkit服特有]</sup> | Bukkit的帮助配置文件<br/>_**这玩意你是用不到的，当摆设就行了**_ |
| 📝 | _permissions.yml_<sup>[Bukkit服特有]</sup> | 配置玩家和玩家组（用户组）的权限用的配置文件<br/>_**这玩意你是用不到的，当摆设就行了**_ |
| 📝 | spigot.yml<sup>[Spigot及其分支特有]</sup> | Spigot服务器引擎的配置文件 |
| 📝 | usercache.yml | 存储服务器所知的玩家UUID与其用户名之间的映射<br/>用于确保即使在Mojang名称更改政策下也能正确识别玩家 |
