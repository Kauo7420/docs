上面的`.gz`后缀名的文件就是日志文件的压缩包，就是之前的日志的压缩包，而`latest.log`就是最后一次开服时保存的日志了。

::: warning 防歧义（理解误区）

`.gz`后缀名的文件是一种压缩包文件的后缀名，不是Minecraft服务器专用的日志压缩包的文件后缀名。

:::

发现上面的格式规律了吗？没错，它们的格式是：`年-月-日-第几次开服.log.gz`

::: tip 你知道吗

为了节省空间，Minecraft的开发者设定了这么一个功能：之前的日志都会以压缩包的形式存在于`logs`文件夹中，当你启动MC时，上一次的日志（latest.log）会被打包压缩成`.log.gz`文件后缀名的压缩包，并标注时间信息，这样不仅可以节省日志文件的空间占用，还可以让服主/玩家更方便地去寻找所需要的日志。

:::
